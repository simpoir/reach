ABOUT
=====

Reach is the "just connect to that host and I don't care how", ssh client.
It can also be described as: "Dang! I forgot to pass tunnel args.
Nevermind, I'm using Reach". Yes it is that cool.


REQUIREMENTS
============

 * python 2.6

   * argparse

 * pexpect

 * ssh client
